Here are 15+ perfect, creative, professional project title suggestions for your art-sharing platform based on the theme ARTIFY – A Creative Artwork Showcase Platform.
All titles fit the vibe of an online gallery, artist community, and modern UI concept.

## Features

- Browse hundreds of public artworks uploaded by artists.
- Filter by category, artist name, or title using smart search.
- View trending artworks, featured collections, and recent uploads.
- Each user gets a personal My Gallery page.
- Artists can upload artworks with images, medium/tools, dimensions, price, and visibility.
- Users can update or delete their artworks.


## Tech Stack

**Client:** React, Tailwind CSS, DaisyUI, Firebase Authentication

**Server:** Node.js, Express.js, MongoDB, CORS
